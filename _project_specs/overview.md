# Project Overview

## Vision

打造台灣首個具備絕對公信力的運動教練評價平台「達達運動」。
透過零信任架構與 AI 輔助驗證，解決進階訓練者「試錯成本極高」的痛點。

## Goals

- [ ] 建立 AI 輔助 PoS 驗證的評價提交流程 (Conversational Review Flow)
- [ ] 實作漸進式信任評分系統 (Progressive Trust Scoring)
- [ ] 教練認領與去重機制 (IG Webhook + OTP)
- [ ] 雙軌制評分顯示 (歷史總評 + 近半年)
- [ ] Pro 方案 SaaS 訂閱 (精選釘選、版面客製、流量分析)

## Non-Goals

- 不介入主觀教學品質仲裁
- 不允許付費刪除或隱藏負評
- 不建立社群聊天功能 (MVP)

## Success Metrics

- 教練認領率 > 30%
- 評價完成率 (含 PoS 上傳) > 60%
- 惡意評價攔截率 > 95%
